How will Semantic Web help Governments?

No matter where you fall in debates over free software or DRM, there's one type of information that is unarguably meant to be free, and that's information about our government.
The more knowledge citizens have about government the better.
So how can we use the Semantic Web to make it easier to get that knowledge, and to foster civic participation?


Every government stores data about its people and its institutions.
This data can vary from high school enrollment to proposed bills.
If this data is stored keeping in mind an Ontology and using RDFa, users of the database will have much more power at their command.
The data will be connected by relationships.
It is these relationships between data that will enable users to perform even more accurate searches.



Scientists using Semantic Web enabled Databases

Tim Berners Lee vigorously campaigned for Open Data and Linked Data in a TED talk given by him.
Most research is conducted separately and the data is stored in different locations.
If this data would be made Semantic Web enabled using Ontologies and RDFa, Scientists will be able to get an exponentially higher degree of accuracy.
