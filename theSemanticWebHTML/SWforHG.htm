How will Semantic Web help Governments?

No matter where you fall in debates over free software,
there's one type of information that is unarguably meant to be free,
and that's information about our government.

The more knowledge citizens have about government the better.
So how can we use the Semantic Web to make it easier to get that knowledge, and to foster civic participation?


Every government stores data about its people and its institutions.
This data can vary from high school enrollment to proposed bills.
If this data is stored keeping in mind an Ontology and using RDFa, users of the database will have much more power at their command.

The data will be connected by relationships.
It is these relationships between data that will enable users to perform even more accurate searches.


Benefits of Open Data and Semantic Web - European Data Portal

European Data Portal (EDP)

This is an example of an initiative towards a world of Open Linked Data.
There are numerous datasets available online from various countries.
Furthermore, anyone can run SPARQL queries on this website.

Example US website : govtrack.us

The US government has been posting relevant information online, but in scattered locations.
For instance, legislation is posted in one place and votes on the very same legislation in another.
Gathering the information in one place and in a common format gives rise to new ways of mixing the information together.

Questions you can ask:

1) What's the population of each senator's state?
2) Which state has more politicians who have proposed bills for Topic X

This gives immense power to the people of the nation.
The dataset can be manipulated for various uses.
The most important factor is that results are more accurate than current search engines.

Scientists using Semantic Web enabled Databases

Tim Berners Lee vigorously campaigned for Open Data and Linked Data in a TED talk given by him.
Most research is conducted separately and the data is stored in different locations.
If this data would be made Semantic Web enabled using Ontologies and RDFa, Scientists will be able to get an exponentially higher degree of accuracy.
A scientist may ask : "What proteins are involved in signal transduction and also related to pyramidal neurons?"
